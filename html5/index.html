HTML5引入了许多新元素和属性

  • 新的语义化元素： <header> , <footer> , <section>
  • 表单2.0： 改进了Web表单， 为 <input> 标签引入了新的属性
  • 持久的本地存储： 不用再通过第三方实现了
  • WebSocket: 用于Web应用程序的下一代双向通信技术
  • 服务器推送事件：HTML5 引入了从 Web 服务器到 Web 浏览器的事件，也称为服务器推送事件（SSE）
  • Canvas: 支持用 Javascript 以编程的方式进行二维绘画
  • 音频和视频: 在网页中嵌入音频或视频无需借助第三方插件
  • 地理定位: 用户可以选择与我们的网页共享他们的地理位置
  • 微数据: 允许创建 HTML5 之外的自定义词汇表，以及使用自定义语义扩展网页
  • 拖放: 把同一网页上的条目从一个位置拖放到另一个位置
