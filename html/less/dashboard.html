Less 示例

LESSCSS是一种动态样式语言，属于CSS预处理语言的一种，它使用类似CSS的语法，为CSS的赋予了动态语言的特性，如变量、继承、运算、函数等，更方便CSS的编写和维护。

LESSCSS可以在多种语言、环境中使用，包括浏览器端、桌面客户端、服务端。

LESSCSS的使用是很容易的，首先，使用你最常使用的代码编辑器，按LESSCSS的语法规则写好.less文件，接下来，使用编译工具它编译成.css，最后再引入页面即可。

LESSCSS也可以不经编译，直接在浏览器端使用。

2种引入方式

GUI编译工具

koala(Win/Mac/Linux):国人开发的LESSCSS/SASS编译工具。
下载地址：http://koala-app.com/index-zh.html

Node.js库

安装less:npm install -g less

使用lessc来编译.less文件:lessc demo.less demo.css

浏览器端使用

  1. 下载LESSCSS的.js文件，例如less.min.js。
  2. 在页面中引入.less文件
    <link rel="stylesheet/less" href="demo.less" />
  3. 引入第1步下载的.js文件
    <script src="less.min.js"></script>

模式匹配

//LESS
// 只接受dark做为首参
.mixin(dark, @color){
color: darken(@color, 10%);
}

// 只接受light
.mixin(ligth, @color){
color: lighten(@color, 10%);
}

// 接受任意值
.mixin(@_, @color){
display: block;
}

//现在运行如下代码：

.mixin-test{
.mixin(@switch, #888888);
}

//CSS
.mixin-test{
    color:#a2a2a2;
    display: block;

导引表达式

导引中可用的全部比较运算有： > >= = =< <。此外，关键字true只表示布尔真值。 导引序列使用逗号‘,’—分割，当且仅当所有条件都符合时，才会被视为匹配成功。 最后，如果想基于值的类型进行匹配，我们就可以使用is*函式。


常见的检测函式

  • iscolor
  • isnumber
  • isstring
  • iskeyword
  • isurl
  • ispixel
  • ispercentage
  • isem
//LESS
.mixin(@a) when (lightness(@a) >= 50%){
background-color: black;
}

.mixin(@a) when (lightness(@a) < 50%){
background-color: white;
}

.mixin(@a){
color:@a;
}

.class1{
.mixin(#dddddd);
}

.class2{
.mixin(#555555);
}

嵌套规则

注意 & 符号的使用—如果你想写串联选择器，而不是写后代选择器，就可以用到&了. 这点对伪类尤其有用如 :hover 和 :focus.

//LESS
#head{
    color: black;

    .navigation{
        font-size:18pt;
    }

    .logo{
        width:300px;
        &:hover{
        text-decoration: none;
        }
    }
}
或者这样写: 
//LESS
#head           { color: black;
    .navigation { font-size: 18pt }
    .logo       { width: 300px;
        &:hover { text-decoration: none }
    }
}

//CSS
#head { color: black; }
#head .navigation {
    font-size: 18pt;
}
#head .logo {
    width: 300px;
}
#head .logo:hover {
    text-decoration: none;
}
//示例

运算

任何数字、颜色或者变量都可以参与运算. 

//LESS
@base: 5%;
@filler: @base * 2;
@width: 2px;

#test-calc{
    color: #888 / 4;
    background-color: #9acfea;
    height: 100% / 2 + @filler;
    width:100px + 4;
    border:(@width * 2) solid red;
}

//CSS
#test-calc{
    color: #222;
    background-color: #9acfea;
    height: 60%;
    width: 104px;
    border: 4px solid red;
}
//示例
