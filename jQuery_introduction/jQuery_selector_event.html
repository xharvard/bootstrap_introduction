选择器之 属性选择器

jQuery( "[attribute|='value']" )

描述: 选择指定属性值等于给定字符串或以该字符串为前缀（该字符串后跟一个连字符“-” ）的元素。

$('a[hreflang|="en"]')
Some text Some other text will not be outlined

jQuery( "[attribute*='value']" )

描述: 选择指定属性具有包含一个给定的子字符串的元素。（选择给定的属性是以包含某些值的元素）

$('input[name*="man"]')

jQuery( "[attribute~='value']" )

描述: 选择指定属性用空格分隔的值中包含一个给定值的元素。

$('input[name~="womn"]')

jQuery( "[attribute$='value']" )

描述: 选择指定属性是以给定值结尾的元素。这个比较是区分大小写的。

$('input[name$="letter"]')

jQuery( "[attribute='value']" )

描述: 选择指定属性是给定值的元素。

$('input[value="Hot Fuzz"]')

jQuery( "[attribute!='value']" )

描述: 选择不存在指定属性，或者指定的属性值不等于给定值的元素。

$('input[name!="newsletter"]')

jQuery( "[attribute^='value']" )

描述: 选择指定属性是以给定字符串开始的元素

$('input[name^="news"]')

jQuery( "[attribute]" )

描述: 选择所有具有指定属性的元素，该属性可以是任何值。

$('div[id]')
[click me]no id
[click me]with id
[click me]has an id
[click me]nope

jQuery( "[attributeFilter1][attributeFilter2][attributeFilterN]" )

描述: 选择匹配所有指定的属性筛选器的元素

$('input[id][name$="value"]')

选择器之 基础选择器

jQuery( "*" )

描述: 选择所有元素.

$("*")
DIV
SPAN

P

jQuery( ".class" )

描述: 选择给定样式类名的所有元素。

$(".myClass")
div class="notMe"
div class="myClass"
span class="myClass"
span class="myclass"

jQuery( "element" )

描述: 根据给定（html）标记名称选择所有的元素。

$("address")
add1
add2

add3

jQuery( "#id" )

描述: 选择一个具有给定id属性的单个元素。

$("#divtest1")

id="divtest1"

id="divtest2"

jQuery( "selector1, selector2, selectorN" )

描述: 将每一个选择器匹配到的元素合并后一起返回。

$("div,span,p.ptest1")
div

p class="ptest1"

p class="ptest2"

span
