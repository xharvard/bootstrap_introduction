特效之 基本特效

.hide( [duration ] [, complete ] )

描述: 隐藏匹配的元素。

所有jQuery效果，包括.hide(),都能通过设置jQuery.fx.off = true全局的关闭，效果等同于持续时间设置为0。
$('.target').hide();

.show( [duration ] [, complete ] )

描述: 显示匹配的元素

$('.target').show();

.toggle( [duration ] [, complete ] )

描述: 显示或隐藏匹配元素。

$('.target').toggle();
